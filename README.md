## ğŸ“¸ How It Works

1. Place your cursor on a Composable function (e.g., `Text`, `Button`).
2. Press `Alt + Enter`.
3. Select **"Wrap with Row/Column/Box/Lazy..."**.
4. Choose your wrapper from the popup menu.
