Wraps Jetpack Compose elements with layout containers like Row, Column, Box, and Lazy layouts.

This inspection detects Composable call expressions and provides quick-fix actions to encapsulate them within common layout wrappers, handling both code structure and necessary imports automatically.

How to use:

  • Place the caret on a Composable (e.g., Text).
  • Press Alt+Enter.
  • Select "Wrap with..." and choose your layout.

Example:


// Before
Text("Hello")

// After wrapping with Column
Column {
    Text("Hello")
}
