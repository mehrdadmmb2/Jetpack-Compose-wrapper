Wraps a single Jetpack Compose UI element with a selected layout composable.
This intention allows quick wrapping of common composables like Text, Button, or Image with Row, Column, Box, LazyRow, or LazyColumn to build layouts faster.
Place the caret on the composable name (e.g., Text), press Alt+Enter, and select the desired wrapper from the dialog.
textText("Hello World") textColumn { Text("Hello World") }
This intention is available only inside @Composable functions and on supported composables. For more options or issues, check the plugin settings.
